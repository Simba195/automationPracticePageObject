## General info
This is a project created in order to practice writing automated functional tests. Portal that was automated in this project is http://automationpractice.com.
	
## Technologies and setup
Project is created with:
* Java 11
* Maven
* Selenium WebDriver
* jUnit 5.x
* Page object pattern
* Page factory
